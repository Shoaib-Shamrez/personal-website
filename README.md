# My Personal Portfolio Website

This is my personal portfolio website built with Next.js. It showcases my projects, resume, skills, and contact information.

## Features

- Built with Next.js, React, and TypeScript
- Responsive design with SCSS
- Sections for About, Projects, Resume, Stats, and Contact
- Easy to customize and update

## Dependencies

- Node.js >= v20

## Setup and Running

1. Clone the repository:

   ```bash
   git clone <your-repo-url>
   cd personal-site
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

   The site will be available at http://localhost:3000/

## Building for Production

To build the site for production:

```bash
npm run build
```

The static files will be in the `out/` directory.

## Customization

- Update personal information in `src/data/`
- Replace images in `public/images/`
- Modify styles in `src/static/css/`
